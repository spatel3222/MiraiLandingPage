## Brand Voice Guidelines

### Writing Style
- Direct and decisive, not flowery
- Technical but accessible
- Professional yet conversational
- Slightly contrarian but constructive
- No buzzwords or press release language
- Write for smart executives
- Personal and authentic
- Outcome-focused over theory

### Language Principles
- Short sentences (10-12 words optimal)
- Active voice
- Concrete examples over abstractions
- Numbers and metrics when available
- "You" focused messaging
- Clear value in every paragraph

### Tone Variations by Content Type
- **Technical posts**: Authoritative yet approachable
- **Transformation stories**: Encouraging and relatable
- **Contrarian takes**: Respectful but firm
- **Tutorials**: Patient and detailed

### Avoid
- Em dashes (—)
- Corporate jargon
- Generic AI commentary
- Theoretical concepts without practical application
- Long, complex sentences

### Embrace
- Short, punchy sentences
- Real metrics and examples
- Practical frameworks
- Personal experiences
- Actionable insights

## Post Creation Process

### Step 1: Hook Development
- Start with a bold statement or surprising insight
- Use pattern interrupts
- Make it personally relevant to the reader
- Two sentences maximum

### Step 2: Value Delivery
- One core insight per post
- Support with personal experience
- Include specific examples
- Add data when available

### Step 3: Engagement Optimization
- Ask a specific question at the end
- Create natural comment opportunities
- Include a clear next action
- Reference the AI PM Assessment when relevant

### Step 4: Format Selection
- Default to carousel for complex topics
- Use text + image for quick insights
- Create polls for community temperature checks
- Design infographics for data-heavy content

## Carousel Creation Guidelines

### Structure
- Slide 1: Hook with clear value proposition
- Slides 2-3: Problem/context setup
- Slides 4-7: Solution/framework/steps
- Slide 8: Results/metrics
- Slide 9: Call to action with question

### Design Principles
- One key point per slide
- Use bullet points liberally
- Include icons/emojis for visual interest
- Maintain consistent formatting
- Keep text under 50 words per slide
