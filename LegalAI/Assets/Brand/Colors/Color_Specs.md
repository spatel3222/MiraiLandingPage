# Mirai360.ai Color Specifications

## Primary Authority Colors

### Navy Primary
- **Hex**: #1B365D
- **RGB**: 27, 54, 93
- **CMYK**: 71, 42, 0, 64
- **Usage**: Authority, trust, legal excellence, primary text, headers

### Navy Light  
- **Hex**: #405A7A
- **RGB**: 64, 90, 122
- **CMYK**: 48, 26, 0, 52
- **Usage**: Readable text, accessibility, body text

### Tech Accent
- **Hex**: #2E86C1
- **RGB**: 46, 134, 193
- **CMYK**: 76, 31, 0, 24
- **Usage**: Innovation, progress, CTAs, highlights

### Neutral Base
- **Hex**: #FAFBFC
- **RGB**: 250, 251, 252
- **CMYK**: 1, 0, 0, 1
- **Usage**: Clean backgrounds, breathing room

## Supporting Colors

### Clean White
- **Hex**: #FFFFFF
- **RGB**: 255, 255, 255
- **CMYK**: 0, 0, 0, 0
- **Usage**: Clarity, transparency, main backgrounds

### Success Green
- **Hex**: #28B463
- **RGB**: 40, 180, 99
- **CMYK**: 78, 0, 45, 29
- **Usage**: Growth, success metrics, positive indicators

### Professional Gray
- **Hex**: #566573
- **RGB**: 86, 101, 115
- **CMYK**: 25, 12, 0, 55
- **Usage**: Balance, sophistication, secondary text

## Accessibility Standards

All color combinations meet WCAG 2.1 AA standards:
- Navy Primary on White: 10.1:1 contrast ratio
- Tech Accent on White: 4.8:1 contrast ratio
- Navy Light on White: 7.2:1 contrast ratio
- Professional Gray on White: 4.5:1 contrast ratio

## Usage Guidelines

### Primary Combinations
- Navy Primary + Clean White (high impact)
- Tech Accent + Clean White (innovation focus)
- Navy Light + Neutral Base (readable content)

### Accent Usage
- Success Green: Use sparingly for metrics and positive outcomes
- Tech Accent: CTAs, links, and innovation highlights only
- Professional Gray: Supporting text and subtle elements

### Avoid
- Using more than 3 colors in single composition
- Navy Primary + Tech Accent without white balance
- Low contrast combinations
- Success Green for non-success content
