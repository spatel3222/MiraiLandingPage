## Sizing Specifications

### Primary Logo Minimum Sizes
- **Digital**: 120px width minimum
- **Print**: 1.5 inches width minimum
- **Recommended**: 160px+ for digital clarity

### Monogram Minimum Sizes
- **Digital**: 24px square minimum
- **Print**: 0.3 inches square minimum
- **Social Media**: 400x400px recommended

## Clear Space Requirements

### Primary Logo
- **All Sides**: 0.5x logo height
- **Example**: If logo is 40px tall, use 20px clear space

### Monogram
- **All Sides**: 0.25x symbol height
- **Example**: If monogram is 80px, use 20px clear space

## Acceptable Uses

### ✅ DO
- Use provided logo files without modification
- Maintain proportional scaling
- Ensure adequate clear space
- Place on appropriate backgrounds
- Use standard or light versions as specified
- Maintain original colors when possible

### Background Guidelines
- **Navy Background**: Use light logo version
- **White/Light Background**: Use standard logo
- **Photo Background**: Ensure sufficient contrast
- **Colored Background**: Test readability carefully

## Prohibited Uses

### ❌ DON'T
- Stretch, distort, or modify proportions
- Change colors (except approved variations)
- Add shadows, effects, or outlines
- Place on busy/cluttered backgrounds
- Use outdated logo versions
- Rotate or skew the logo
- Change font or spacing
- Use as watermark or texture

### Common Mistakes to Avoid
- **Low Resolution**: Pixelated or blurry logos
- **Poor Contrast**: Logo not readable on background
- **Crowded Layout**: Insufficient clear space
- **Wrong Version**: Standard logo on dark background
- **Scaling Issues**: Non-proportional stretching

## File Formats & Usage

### SVG (Scalable Vector Graphics)
- **Best For**: Web, digital applications
- **Advantages**: Infinite scalability, small file size
- **Usage**: Websites, digital presentations, apps

### PNG (Portable Network Graphics)
- **Best For**: Web, presentations, documents
- **Available**: High-res (300dpi), Web (72dpi)
- **Usage**: Social media, email, documents

### EPS (Encapsulated PostScript)
- **Best For**: Print, professional design
- **Usage**: Business cards, letterheads, signage
- **Requirement**: Vector-capable software

## Application-Specific Guidelines

### Digital Applications
- **Websites**: Use SVG for scalability
- **Social Media**: PNG format, appropriate sizes
- **Email**: PNG format, 72dpi resolution
- **Apps**: Multiple resolutions (1x, 2x, 3x)

### Print Applications
- **Business Cards**: EPS or high-res PNG (300dpi)
- **Letterheads**: Vector format preferred
- **Signage**: Vector format required
- **Marketing Materials**: 300dpi minimum

### Social Media Specifications
- **Facebook Profile**: 400x400px PNG
- **LinkedIn Profile**: 400x400px PNG
- **Twitter Profile**: 400x400px PNG
- **Instagram Profile**: 400x400px PNG
