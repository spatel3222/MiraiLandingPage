# CRTX Website

Enterprise AI consulting services website for CRTX - offering AI transformation, mentoring, and investment services.

## Overview

CRTX provides three core services:
- **AI Consulting**: Enterprise AI transformation with Fortune 1 proven strategies
- **AI PM Mentoring**: Career transformation for product managers entering the AI space
- **VC Investment**: Deep tech investment in partnership with IIMA Ventures

## Tech Stack

- HTML5
- Tailwind CSS (via CDN)
- Vanilla JavaScript
- Formspree (for contact forms)
- Airtable API (for data collection)

## Structure

```
crtx-website/
├── index.html                 # Homepage
├── about.html                 # About page
├── consulting.html            # AI consulting services
├── mentoring.html             # AI PM mentoring services
├── investment.html            # VC investment services
├── ai-pm-assessment.html      # Interactive AI PM assessment tool
├── thank-you.html             # Form submission confirmation
├── CNAME                      # Custom domain configuration
└── README.md                  # This file
```

## Features

- Responsive design for all devices
- Interactive AI PM assessment with scoring
- Contact forms with Airtable integration
- Service-specific landing pages with unique branding
- LinkedIn content examples and insights

## Setup

1. Clone the repository
2. No build process required - static HTML files
3. Update API keys in relevant files:
   - Formspree endpoint in contact forms
   - Airtable API key in assessment tool

## Deployment

The site is designed to be deployed on GitHub Pages:

1. Push to GitHub repository
2. Enable GitHub Pages in repository settings
3. Set custom domain in CNAME file
4. Site will be live at crtx.in

## Contact Forms

All contact forms submit to:
- Formspree for email notifications
- Airtable for CRM integration
